1. API 기본 정보

메서드 요청 URL 출력 포맷
GET http://35.173.220.182:8080/ JSON

2. 요청 변수

요청 변수 타입 필수 여부 기본값 설명
date string Y - 검색을 원하는 연도와 달, YYYY-MM-DD 형식을 입력한다.
area string Y - 검색을 원하는 지역, 17개 도시의 검색이 가능하다.

3. 출력 결과

필드 타입 설명
Administrative_Area string 행정 구역명
Observation string 관측 일(YYYY-MM-DD)
P_pv Float 일간 관측된 태양광 에너지 발전량
P_wind Float 일간 관측된 풍력 에너지 발전량
P_load Float 일간 관측된 전력 발전량
Optimal_prompt1 Float 일간 태양광, 풍력 발전 최적 비율 중 태양광 발전 비율
Optimal_prompt2 Float 일간 태양광, 풍력 발전 최적 비율 중 풍력 발전 비율
Optimal_cost Integer 일간 최적의 비율로 생산되는 전력 발전 설비 비용
Optimal_soc Float 일간 최적의 배터리 충전 상태
Optimal_p_wind Float 일간 최적의 풍력 에너지 발전량
Optimal_p_dummy Float 일간 배터리 충전 초과 후 잉여전력
Optimal_p_pv Float 일간 최적의 태양광 에너지 발전량
Optimal_p_battery Float 일간 최적의 비율로 생산되어 충전된 배터리 양
Optimal_p_load Float 일간 최적의 전력 발전량
Optimal_p_dg Float 일간 최적의 디젤발전기 발전용량
